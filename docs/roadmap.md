# Roadmap

## M0 Prototype

- Bridge mock mode.
- Firmware renders mock data from the bridge.
- LILYGO T-Display-S3 flash path works.

## M1 Proxmox MVP

- Native Proxmox API provider.
- Multiple separate Proxmox installs.
- Host overview, guest summary, pinned guests.
- Basic storage and memory alerts.
- Docker Compose setup.

## M2 Product Quality

- Captive portal polish.
- Web flasher with release binaries.
- Bridge page with better diagnostics.
- Screenshots and photos.
- LXC guide and binary releases.

## M3 Community

- Community-scripts-style LXC installer.
- Additional board profile system.
- Optional Pulse provider if user demand is real.
